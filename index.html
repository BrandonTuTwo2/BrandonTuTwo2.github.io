side projects

Here are some of the side projects that I have worked on, I will keep this updated with different side projects as time goes on~

Will It Rain

A web application/website that displays when it will rain in the next 7 days by converting an inputted address to a latitude and longitude (via nominatim) and getting a forcast from 7timer in JSON format and displaying that information to the user.

XKCD Comic Viewer

A web application/website that displays the xkcd comics (using its API) and various functions related to navigating the comics.

Rep Set Counter

An android app that tracks how long a rep should be, how long a rest should be and how many reps to do, useful when working out doing timed reps and doing HIIT workouts.

Word Replacer

A C# program that takes in a text file and replaces the user entered word with another word.
